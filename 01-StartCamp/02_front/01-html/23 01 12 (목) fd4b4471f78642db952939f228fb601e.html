    • <태그명>, <h2> <div> 브라우저는 태그명을 보고 어떤 의미를 갖는지 판단(마크업 기능)

  • Attribute
    • 시작태그에만 붙을 수 있는것이 attribute(속성)임
    • 이름 = ‘값’
    • ex) <div id = “a”>
    • 꼭 하나가 아니라 없을수도 있고 여러개일 수도 있음
    • ex) <div id = “a” class = “b”>
    • 속성이 필수는 아니나 속성을 주기 위해서는 시작태그에 준다는 것을 기억
    • 시작태그이자 종료태그인 것들에 붙여도 된다.
